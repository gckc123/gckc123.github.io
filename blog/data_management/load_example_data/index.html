Load Example Data

20 Sep, 2020 | 1 minute read
Share this: Twitter Facebook

Tag: Data Management

StatsNotebook comes with a rich data library for learning statistics. To load an example dataset,

  1. Click the Menu icon     at the top
  2. Click “Open”
  3. Select “Example Data” from the menu

  1. Click on the example data we want to load.
