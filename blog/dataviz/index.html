Animated plots

The tutorial is based on R and StatsNotebook, a graphical interface for R. Animated plots can be created using the gganimate package, an extension of the ggplot2 package. We will use the built-in Gapminder dataset for scatterplot example and the COVID-19 data for the line graph, world map and barchart.
