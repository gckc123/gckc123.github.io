Line chart

The tutorial is based on R and StatsNotebook, a graphical interface for R. Line chart can be used to show the evoluation of numeric variables. StatsNotebook uses geom_line() from ggplot2 to build line chart. We use the built-in Human Development Index dataset in this example. This dataset can be loaded into StatsNotebook using instruction here. This is a data from the United Nations Development Programmes (UNDP). We will use line chart to visualise the evolution of Human Development Index from 1990 to 2018 across six regions. Read more
